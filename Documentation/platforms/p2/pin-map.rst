P2-EC32MB pin map
=================

Status: DRAFTED from module guide assumptions, HIL-REQUIRED.

P0-P39 are application Smart Pins; P38/P39 also drive buffered LEDs. P40-P57 are PSRAM. P58/P59 are shared MISO/MOSI. P60/P61 exchange clock/select roles between flash and microSD. P62/P63 are console TX/RX. Reserved pin ownership is enforced by the draft pin manager.
