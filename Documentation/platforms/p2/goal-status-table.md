# Propeller 2 NuttX goal status

Updated 2026-07-13 during the final hardware-in-the-loop campaign.

Status meanings:

- **PASS** — supported by a preserved physical-board artifact.
- **PARTIAL / IN PROGRESS** — useful functionality works, but the full acceptance gate is not complete.
- **BLOCKED / NOT IMPLEMENTED** — a precise missing fixture or dependency remains; this is not a support claim.
- **DEFERRED / OUT OF SCOPE** — deliberately excluded from this flat-UP completion goal; unsupported behavior is still stated explicitly.

Hardware: P2-EC32MB Rev B at `/dev/cu.usbserial-P97cvdxp`. Installed digital loopbacks are direct shorts P0–P1, P2–P3, and P6–P7. Digital tests configure the receiving end before driving one end only. The former P4–P5 direct jumper has been replaced by the requested resistive/capacitive analog fixture. A BMP180 I²C sensor is installed with P24 as SDA and P25 as SCL.

## Finish-line summary

The stable flat-UP NuttX core, NSH, digital and analog Smart Pin baseline, BMP180 I²C host path, flash/SmartFS, independent DTR-reset flash boot, microSD/FAT, shared storage arbitration, and explicit PSRAM service are working on the physical board.

The complete applicable OSTest campaign, post-freeze ABI matrix, host/static checks, artifact indexing, and final documentation are green. The flat-UP goal is complete. True power-cycle/power-loss, card-absent behavior, the printed boot-switch label, and fixture-dependent AIO/multi-user rows remain explicit non-gating evidence gaps. SMP is deliberately out of scope and does not block this flat-UP goal.
